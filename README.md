# Spring Boot 3.x WebFlux Demo Project

这是一个基于Spring Boot 3.x和WebFlux的响应式Web应用示例项目。

## 技术栈

- Spring Boot 3.2.3
- Spring WebFlux
- Project Reactor
- Lombok
- Spring Boot Actuator

## 系统要求

- JDK 17+
- Maven 3.6+

## 如何运行

1. 克隆项目到本地
2. 在项目根目录执行：`mvn spring-boot:run`
3. 访问 http://localhost:8089

## 项目特性

- 基于响应式编程模型
- 非阻塞式Web框架
- 内置健康检查和监控（Actuator）
- 支持响应式流
